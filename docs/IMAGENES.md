# Imágenes de la web

La web pública usa imágenes reales y no ilustraciones para los bloques visuales.

## Imagen de Acompañamiento familiar

Archivo usado:

```text
assets/familia-acompanamiento.png
```

Ubicación en `index.html`:

```html
<section id="familias"> ... <img src="assets/familia-acompanamiento.png" alt="Acompañamiento familiar" /> ... </section>
```

La imagen fue provista por el usuario y reemplaza la referencia externa anterior.

## Regla

No usar vectores ni placeholders en los bloques institucionales finales. Si se reemplaza una imagen, mantener el nombre del archivo o actualizar `index.html` con ruta local dentro de `assets/`.
